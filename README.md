# The Description
There are 12 scatterplots, one for each soil type and quarter. The scatterplots show the counts of the different bacteria compositions within the chosen soil type during the chosen quarter. There are 40,324 different bacteria compositions.
There are 3 grouped bar charts. Each chart is for a different quarter. The four soil types are the grouped bars with the bins being the top three bacteria compositions recorded. The counts for those compositions is the y-axis of the charts.
These plots were all made with Python3 in Jupyter Notebook. Within the code, the soil types are abbreviated. BS is for base soil, GL is for grasslands, P is for Poplar Plantation, and MF is for mixed forest.
